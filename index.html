Learning Journey
In March, I completed a CodeFirstGirls Kickstarter course in JavaScript, and what a ride that was!! Now at the end of May, I have also completed the CodeFirstGirls Kickstarter course in Python.

Although in the past I dabbled with web design using programs like Dreamweaver or Wix, in the last 5 months I've learned JavaScript, Python, HTML, CSS, CodePen, PyCharm, Replit, Glitch, Slack, GitHub (still to be determined). Which thinking back now is pretty amazing considering the actual instructional part of the each course totalled 12 hours with the additional last 4 hours being about our final projects.

I've spent every waking spare minute of my day coding. I've gone to bed counting code instead of sheep, I've woken up earlier than I needed to, just so I could do some coding before I started work.
