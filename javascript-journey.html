Assignment 4 : Room Colour Customizer (Javascript, HTML & CSS)
This assignment was a little more challenging mostly because my idea required the use of an image and that alone caused problems as I don’t have a hosting account and also because I wanted the image to overlay a background that changed colour. I could have just changed my idea, but instead I began researching for a way to store images online rather than in a local directory.

Initially I found and used a way to store it on my Google Drive, but of course Google are not a hosting site, so I knew it wasn’t a stable option, but it worked as an interim. Once I had the rest of my code working, I then went back to researching for a better option. I eventually found a recommendation on Stack Exchange for Cloudinary which has been a life saver for my future and final projects too.

I also tried a version using columns but that proved just a little bit too difficult for me at this time as I wasn’t happy with how one of the images was behaving, so I amended my design and went back to a single column layout centred in the middle of the page. The other problem with this assignment was the instructor had problems with connection this week and wasn’t able to finish the class. We were actually given a pass with our homework, but I decided to forge ahead and finish the studies myself and submit my assignment anyway.

Assignment Brief : Create a web app that takes input from the user and updates the web page based on the input. You should: + Get input from the user on a web page + Make changes to the HTML and/or CSS using JavaScript + Use an event to trigger a change to a web page

Bonus : + Add HTML elements to a web page using the DOM + Remove HTML elements from a web page using the DOM + Research and use other DOM events such as onmouseover + Use arrays, loops and/or conditional logic in JavaScript

FINAL COURSE PROJECT : Car Customizer (Javascript, HTML & CSS)
For my final project, I wanted to expand further on what I had achieved in my previous projects.

I started off by getting the actual functions to work, which I achieved in version 12, but then I wanted to improve on the design and completely lost it. I just couldn’t replicate the last part of the function again with the original car image being replaced with the patterned car image. Eventually after version 29, I asked my instructor for help. Turns out it only needed a minor adjustment.

While I was waiting for him to get back to me, I also realised the way I had set up the columns and rows was messy and causing problems, so I decided to draw a line under it all and start from scratch using a simpler method of setting up a grid.

After that, it was just a matter of applying the changes my instructor had given me and then figuring out how to do a form and a pop up.

This whole project was a massive learning curve: HTML, CSS and JavaScript. But I am extremely happy with what I have achieved considering at this point, I had only had 12hrs of class time.
